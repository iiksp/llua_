扩展CRT

我们知道，中国剩余定理是用来解同余方程组

⎧⎩⎨⎪⎪⎪⎪⎪⎪x≡c1(mod m1)x≡c2(mod m2)…x≡cr(mod mr){x≡c1(mod m1)x≡c2(mod m2)…x≡cr(mod mr)

但是有一个非常令人不爽的事情就是它要求m1,m2…,mrm1,m2…,mr两两互素

如果某个毒瘤出题人偏要求它们部互素呢？

其实也有解决的办法

就是把出题人吊起来干一顿用扩展中国剩余定理

扩展中国剩余定理跟中国剩余定理没半毛钱关系，一个是用扩展欧几里得，一个是用构造

 

首先我们还是从简单入手，考虑一下如果同余方程组只有两个式子的情况

x≡c1(mod m1)x≡c2(mod m2)x≡c1(mod m1)x≡c2(mod m2)

将两个式子变形

x=c1+m1k1x=c2+m2k2x=c1+m1k1x=c2+m2k2

联立

c1+m1k1=c2+m2k2c1+m1k1=c2+m2k2

移项

m1k1=c2−c1+m2k2m1k1=c2−c1+m2k2

我们用(a,b)(a,b)表示a,ba,b的最大公约数

在这里需要注意，这个方程有解的条件是

(m1,m2)|(c2−c1)(m1,m2)|(c2−c1)，因为后面会用到(c2−c1)(m2,m1)(c2−c1)(m2,m1)这一项，如果不整除的话肯定会出现小数。

对于上面的方程，两边同除(m1,m2)(m1,m2)

m1k1(m1,m2)=c2−c1(m1,m2)+m2k2(m1,m2)m1k1(m1,m2)=c2−c1(m1,m2)+m2k2(m1,m2)

m1(m1,m2)k1=c2−c1(m1,m2)+m2(m1,m2)k2m1(m1,m2)k1=c2−c1(m1,m2)+m2(m1,m2)k2

转换一下

m1(m1,m2)k1≡c2−c1(m1,m2)(mod m2(m1,m2))m1(m1,m2)k1≡c2−c1(m1,m2)(mod m2(m1,m2))

此时我们已经成功把k2k2消去了。

同余式两边同除m1(m1,m2)m1(m1,m2)

k1≡inv(m1(m1,m2),m2(m1,m2))∗(c2−c1)(m1,m2)(modm2(m1,m2))k1≡inv(m1(m1,m2),m2(m1,m2))∗(c2−c1)(m1,m2)(modm2(m1,m2))

inv(a,b)inv(a,b)表示aa在模bb意义下的逆元

k1=inv(m1(m1,m2),m2(m1,m2))∗(c2−c1)(m1,m2)+m2(m1,m2)∗yk1=inv(m1(m1,m2),m2(m1,m2))∗(c2−c1)(m1,m2)+m2(m1,m2)∗y

接下来怎么办呢？这个式子已经化到最简了。。

不要忘了，我们刚开始还有两个式子。我们把k1k1待回去！

x=inv(m1(m1,m2),m2(m1,m2))∗(c2−c1)(m1,m2)∗m1+ym1m2(m1,m2)+c1x=inv(m1(m1,m2),m2(m1,m2))∗(c2−c1)(m1,m2)∗m1+ym1m2(m1,m2)+c1

x≡inv(m1(m1,m2),m2(m1,m2))∗(c2−c1)(m1,m2)∗m1+c1(modm1m2(m1,m2))x≡inv(m1(m1,m2),m2(m1,m2))∗(c2−c1)(m1,m2)∗m1+c1(modm1m2(m1,m2))

此时，整个式子中的元素我们都已经知道了

具体一点，这个式子可以看做是

x≡c(modm)x≡c(modm)

其中

c=(inv(m1(m1,m2),m2(m1,m2))∗(c2−c1)(m1,m2))%m2(m1,m2)∗m1+c1c=(inv(m1(m1,m2),m2(m1,m2))∗(c2−c1)(m1,m2))%m2(m1,m2)∗m1+c1

m=m1m2(m1,m2)m=m1m2(m1,m2)

 

推广一下

我们每次把两个同余式合并，求解之后得到一个新的同余式。再把新的同余式和其他的联立，最终就可以求出满足条件的解

#include<iostream>
#include<cstdio>
#define LL long long 
using namespace std;
const LL MAXN=1e6+10;
LL K,C[MAXN],M[MAXN],x,y;
LL gcd(LL a,LL b)
{
    return b==0?a:gcd(b,a%b);
}
LL exgcd(LL a,LL b,LL &x,LL &y)
{
    if(b==0){x=1,y=0;return a;}
    LL r=exgcd(b,a%b,x,y),tmp;
    tmp=x;x=y;y=tmp-(a/b)*y;
    return r;
}
LL inv(LL a,LL b)
{
    LL r=exgcd(a,b,x,y);
    while(x<0) x+=b;
    return x;
}
int main()
{
    #ifdef WIN32
    freopen("a.in","r",stdin);
    #else
    #endif
    while(~scanf("%lld",&K))
    {
        for(LL i=1;i<=K;i++) scanf("%lld%lld",&M[i],&C[i]);
        bool flag=1;
        for(LL i=2;i<=K;i++)
        {
            LL M1=M[i-1],M2=M[i],C2=C[i],C1=C[i-1],T=gcd(M1,M2);
            if((C2-C1)%T!=0) {flag=0;break;}
            M[i]=(M1*M2)/T;
            C[i]= ( inv( M1/T , M2/T ) * (C2-C1)/T ) % (M2/T) * M1 + C1;
            C[i]=(C[i]%M[i]+M[i])%M[i];
        }
        printf("%lld\n",flag?C[K]:-1);
    }
    return 0;
}
